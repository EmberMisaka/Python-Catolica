# Python para Ciência de Dados e Análise de Dados

## Sobre este Repositório
Este repositório contém materiais didáticos de Python aplicados a **Ciência de Dados** e **Análise de Dados**, desenvolvido para a disciplina da Faculdade Católica.

## Conteúdo Programático

### 1. Fundamentos de Python
- Variáveis e tipos de dados
- Estruturas de controle
- Funções e módulos
- Manipulação de arquivos

### 2. Bibliotecas Essenciais
- **NumPy**: computação numérica
- **Pandas**: manipulação de dados
- **Matplotlib**: visualização
- **Seaborn**: visualização estatística

### 3. Análise de Dados
- Exploração de dados (EDA)
- Limpeza e transformação
- Análise estatística
- Agregação e agrupamento

### 4. Ciência de Dados
- Pré-processamento
- Feature engineering
- Modelagem preditiva
- Avaliação de modelos

## Recursos
- Notebooks Jupyter com exemplos práticos
- Datasets para exercícios
- Scripts de referência
